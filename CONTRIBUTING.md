# Contributing to Scope-AI 🚀

First of all, thank you for considering contributing to Scope-AI! It's people like you that make the open-source community such an amazing place.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub issue tracker.
- Describe the bug and provide steps to reproduce.

### Suggesting Enhancements
- Open an issue to discuss your idea.
- Focus on features that improve AI cost estimation or project architecture planning.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure the code follows the existing style (Next.js/FastAPI).
3. Update the `README.md` if your change adds new functionality.
4. Issue your PR!

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS, shadcn/ui, Recharts.
- **Backend**: FastAPI, LiteLLM.

## Local Development
Run `./run.sh` to start both frontend and backend locally.
